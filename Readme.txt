This Project is developed using simple javascript , css , Html 
Some of the key frameworks that are used are :- Ejs (Embeded javascript) & mongoose (used as an ODM)
The database used in this project is MongoDB.


How to run:-

Run command npm install in your command line in vs-code or as per your desired edittor 

Create a Todo Database connection or you can simply change the connection in Config\config.js .


Thats it you are all good to go 

just type localhost:8000/ in your browser url and boom the app will load
